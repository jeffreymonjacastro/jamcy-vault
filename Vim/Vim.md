# Vim Tutor
+ **h:** Left
+ **j:** Down
+ **k:** Up
+ **l:** Right
+ **w:** Jumps to the begin of the next word
+ **e:** Jumps to the end of the next word
+ **b:** Jumps to the begin of the previous word

# Keybinds
| Description | Neovim | vscode | JetBrains |
| ----------- | ------ | ------ | --------- |

| Down        | k      | k      | k         |
| Up          | l      | l      | l         |
